# BIGWISE Fashion E-commerce

## Project Overview
Premium fashion e-commerce website for BIGWISE, a fashion brand with locations in Ile-Ife & Lagos.

**Current Status:** Complete with messaging feature

## Design & Branding
- **Theme:** Dark purple (#1a1025, #251b35) with premium aesthetic
- **Collections:** Jackets, dresses, streetwear, accessories
- **Vibe:** Bold, modern, smooth animations
- **Typography:** Premium, clean

## Features
- Home page with hero section and featured collections
- Collections page with product grid
- Product detail pages
- Shopping cart functionality
- User authentication
- Wishlist
- Contact page
- **Live chat messaging** - Users can message BIGWISE support via floating chat button

## Project Structure
- Frontend: React + Vite
- Backend: Express.js
- Database: In-memory storage (ready for PostgreSQL)
- Styling: Tailwind CSS + shadcn/ui

## Technical Stack
- react@latest
- express.js
- drizzle-orm
- tailwindcss
- shadcn/ui components
- tanstack/react-query
